**Tweet 2 (problem)**
Most site navigation is built for the org, not the user.

Internal brand names. Org-chart structure. Labels the CEO understands but visitors don't.

The principles to fix this exist. Most designers just don't apply them systematically.

---

**Tweet 3 (what)**
`ia-practitioner` is a Claude skill that applies IA frameworks to real sites.

It knows:
→ The 8 classification schemes + when to use each
→ 10 IA patterns + how to choose
→ How to test labels against "the word in their head"
→ How to structure navigation for multiple user groups

---

**Tweet 4 (live example)**
Live example: we ran it on a real multi-location GI medical group.

5 violations caught immediately:
• Locations by brand name, not geography
• "Treatments" label for symptom-seeking patients  
• Contact buried inside About Us
• "In The News" at top-level nav

All fixed. Nav restructured. Report generated.
